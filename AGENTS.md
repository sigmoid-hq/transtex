## Instructions

### 언어

- 대화: 한국어
- 코드/주석: 영어
- README 등 문서는 기존 언어를 따르거나 사용자가 지정한 언어로 작성
- 요청 시 지정 언어 사용
- 프로젝트 코드를 한 파일에 때려박거나, 모든 파일을 한 디렉토리에 넣지 말 것
- 적절히 디렉토리 구조 잘 설계해서 개발할 것.

### Typescript / Yarn

- 패키지 매니저: `yarn`
- `package.json` 직접 수정 금지, `yarn add/remove` 사용
- 실행은 `yarn <script>` 형태 사용

### Python / UV

- 패키지 매니저로 uv를 사용한다.
- pip 명령어를 사용하거나, 직접 requirements.py나 pyproject.toml을 수정하지 않는다.
- uv add, remove 등 명령어를 사용하여 패키지를 추가/제거하라.
- 코드 실행은 uv run으로 해라.
